# Bash Cheat Sheet

## Print Working Directory

```bash
pwd
```

## List Files

```bash
ls -la
```

## Change Directory

```bash
cd directory_name
```

## Copy Files

```bash
cp source destination
```

## Move Files

```bash
mv source destination
```

## Remove Files

```bash
rm file_name
```

## Create Directory

```bash
mkdir directory_name
```

## Search Text

```bash
grep keyword file.txt
```

## Notes

Bash commands are essential for Linux administration, scripting, and cloud automation.
