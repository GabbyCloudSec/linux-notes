# Networking

## Display Network Interfaces

```bash
ip addr
```

## Display Routing Table

```bash
ip route
```

## Test Connectivity

```bash
ping google.com
```

## DNS Lookup

```bash
nslookup google.com
```

## Display Listening Ports

```bash
ss -tuln
```

## Notes

Networking knowledge is fundamental to Linux administration, cloud computing, and cybersecurity.
