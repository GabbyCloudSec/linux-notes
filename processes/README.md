# Processes

## View Running Processes

```bash
ps aux
```

## Interactive Process Viewer

```bash
top
```

## Kill a Process

```bash
kill PID
```

## Kill a Process by Name

```bash
pkill process_name
```

## View Background Jobs

```bash
jobs
```

## Notes

Processes are programs running on the system. Understanding process management is essential for Linux administration and troubleshooting.
